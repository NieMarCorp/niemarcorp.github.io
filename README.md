# EdgeQuantia

**Plataforma de Ingenieria y Tecnología.**

> ⚠️ **Estado:** En desarrollo activo (Pre-lanzamiento).

## Descripción
Este repositorio aloja la landing page temporal y segura de NieMarCorp. El sitio actúa como punto de contacto inicial mientras se desarrolla la infraestructura principal de la plataforma.

## Seguridad
Este sitio estático implementa políticas de seguridad estrictas (CSP) para prevenir la ejecución de scripts no autorizados y proteger la integridad de la marca durante la fase de construcción.
